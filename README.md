# Role-Based Prompting Personas

This project is a Python exercise focused on role-based prompting for AI.

## What I built
I created three business personas:
- Business Analyst
- Market Researcher
- Strategic Consultant

Each persona includes:
- role definition
- expertise
- communication style
- analytical approach

## What the code does
The Python file defines these personas and includes a validation function to check that each persona is complete.
